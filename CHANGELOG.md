# 更新日志

## 0.1.2 - 2026-07-16

### 修复

- 软件内更新下载完成后改为静默重启安装，不再停在需要二次确认的 NSIS 安装窗口。
- Release Workflow 拉取完整 tag 历史，后续发布页能够稳定读取 annotated tag 中的中文发布说明。

### 测试

- 增加可复跑的线上更新冒烟模式，隔离安装目录、用户数据和 updater 缓存，验证检查、下载、重启安装、数据保留、卸载与清理。

## 0.1.1 - 2026-07-16

### 新功能

- 将 VibeTracker 从手工项目记录器升级为本地项目中枢，打通本地项目导入、Git 扫描、AI 草稿审核、持续同步和一键启动闭环。
- 增加原生目录导入向导、Git 事实库、增量游标、历史基线、提交处理状态和可追溯开发记录。
- 增加 OpenAI-compatible Provider、结构化项目 AI 规则、安全密钥存储、生成范围预览、草稿审核和运行历史。
- 增加结构化 Launch Profile、项目进程管理、就绪检测、日志查看、打开与停止操作。
- 重构 Dashboard、项目画廊、项目详情和设置页，并加入统一 Toast 与后台任务中心。
- 接入 GitHub Release 自动更新检查、下载和重启安装流程。

### 安全与可靠性

- 数据库升级为 17 个顺序迁移版本，迁移前自动备份，并提供事务回滚、完整性校验和损坏恢复。
- 收敛领域 IPC，增加输入、返回值和 sender 校验，以及 CSP、导航和新窗口限制。
- 本地图片改用受控资源协议和有界缩略图缓存，补齐托管资产归属、迁移与孤儿清理。
- Git 命令统一使用参数数组并增加超时、输出限制、异常状态处理和重复同步幂等约束。
- 增加白屏恢复、Git 调度恢复、AI 运行恢复、Launch watchdog 和持久化任务历史。

### 测试

- 增加 146 项单元测试、12 条真实 Electron E2E 流程，以及打包版和 NSIS 安装升级冒烟测试。
- 覆盖迁移保留、Git 幂等、AI Schema、IPC 校验、Launch 生命周期、资产清理、响应式与无 LLM 流程。
